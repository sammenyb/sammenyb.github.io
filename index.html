Highlighted Projects

Security Project Focused on Computer Vision

Attack with Adversarial Stickers. As can be seen, the model predicts red lights as green, which could become dangerous in a real-life scenario.
This project demonstrates the vulnerabilities in a self-driving system, focusing on traffic light detection and classification.

Key Points:
  • Creation of a traffic light detection and classification model using TensorFlow
  • Injection of adversarial stickers to manipulate traffic light recognition
  • Attack on traffic light classification using adversarial stickers
  • Defense strategies to improve model robustness

Robot Swarm algorithm combined with computer vision

This project showcases robots using computer vision together with a Particle Swarm Optimization (PSO) algorithm to collaboratively navigate around obstacles as a group.

Key Points:
  • Implementation of Particle Swarm Optimization (PSO) for robot navigation
  • Training a YOLO model to recognize robots and obstacles
  • Collaborative behavior among multiple robots

Traffic Vehicle Detection and Classification

In this project, we developed an image detection and classification model from scratch, without using pre-trained models. The model was trained to detect and classify different types of vehicles in traffic scenes.

Key Points:
  • Development of a custom image detection and classification model
  • Trained on different types of traffic scenes
  • Tested how well the model finds and classifies vehicles
